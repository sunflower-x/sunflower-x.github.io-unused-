hexo+github创建个人博客

发表于2022-10-29|更新于2022-10-29|博客
|阅读量:
1. 常用工具：
    a. md编辑器：Typora
    b. 代码编辑器：notepad++,vscode,webstrom
2. 环境搭建：
    a. git:版本控制工具
    b. node：js运行环境
    c. npm,cnpm,yarn:包管理工具
3. hexo安装：
    a. 安装：npm install hexo -g
    b. hexo version
    c. 创建博客文件夹；切换到当前目录
    d. 初始化博客：hexo init
4. 发布博客到github
    a. 新建仓库：名字.github.io
    b. 配置config.yml  
    deploy:
      type: git
      repository: https://github.com/sunflower-x/sunflower-x.github.io.git
      branch: main
    c. 切换到博客目录：npm install hexo-deployer-git --save
    d. 清理缓存：hexo clean
    e. 生成文件：hexo g
    f. 部署：hexo d
5. 配置文件说明及常用命令
    a. 注意yml文件的缩进
    b. 新建文章：hexo n "title"
    c. 重新生成：hexo g
    d. 本地预览： hexo s
    e. 上传到github: hexo d
    f. 执行hexo d命令报错
        i. fatal: unsafe repository to add an exception for this directory
            1) https://blog.csdn.net/skybulex/article/details/125542260?spm=1001.2101.3001.6661.1&utm_medium=distribute.pc_relevant_t0.none-task-blog-2%7Edefault%7ECTRLIST%7ERate-1-125542260-blog-125094384.pc_relevant_default&depth_1-utm_source=distribute.pc_relevant_t0.none-task-blog-2%7Edefault%7ECTRLIST%7ERate-1-125542260-blog-125094384.pc_relevant_default&utm_relevant_index=1；
        ii. https://www.myosotis.xin/2021/08/25/hexo-8/
    g. 获取ipv4地址：ping -4 sunflower-x.github.io
6. 主题安装使用:Butterfly
    a. 安装butterfly:npm i hexo-theme-butterfly
    b. 应用主题：修改 Hexo 根目錄下的 _config.yml，把主題改為butterfly
    c. 如果你沒有 pug 以及 stylus 的渲染器，請下載安裝：npm install hexo-renderer-pug hexo-renderer-stylus --save

9. HEXO更新博客后出现404 解决方法
10. 注意：需要在sourece目录下新建一个CNAME文件，无后缀名，把自己的域名填在里面；如果不做这一步，每次hexo d部署到github上以后，对应仓库的域名设置里的域名会被重置回原来的username.github.io，你需要再次设置成你的域名，很麻烦。参考文章
